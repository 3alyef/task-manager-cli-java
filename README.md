# Task Manager CLI (Java)

Gerenciador de tarefas simples feito em Java, com interface de linha de comando (CLI). Projeto feito para praticar fundamentos de Java, como orientação a objetos e estruturas de dados.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Editar tarefas
- Remover tarefas

## Como rodar

```bash
make run

```
